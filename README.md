# ⚙️ Projeto: Engenharia de Recursos (Feature Engineering)

Este repositório contém a aplicação prática de técnicas de **Engenharia de Atributos (Feature Engineering)**, uma etapa fundamental no pipeline de dados para melhorar a performance de modelos de Machine Learning.

## 🎯 Objetivo do Projeto
O objetivo principal deste projeto é transformar dados brutos em recursos (features) informativos e otimizados, facilitando a extração de padrões pelos algoritmos de previsão ou classificação.

## 🛠️ Técnicas e Metodologias Aplicadas
Neste notebook, foram desenvolvidas as seguintes etapas:
- **Análise Exploratória (EDA):** Compreensão da distribuição dos dados e identificação de outliers.
- **Tratamento de Dados Ausentes:** Imputação e remoção de valores nulos.
- **Encoding de Variáveis Categóricas:** Transformação de dados em texto para formatos numéricos (ex: One-Hot Encoding, Label Encoding).
- **Escalonamento e Normalização:** Padronização das escalas numéricas para evitar viés no modelo.
- **Criação de Novas Features:** Geração de novas variáveis a partir das existentes com base em regras de negócio para enriquecer o conjunto de dados.

## 💻 Tecnologias Utilizadas
- **Linguagem:** Python
- **Bibliotecas Principais:** Pandas, NumPy, Scikit-Learn, Matplotlib/Seaborn.
